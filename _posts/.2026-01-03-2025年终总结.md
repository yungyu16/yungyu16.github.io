从来都是糊里糊涂的过日子，很少有冲动来总结自己的生活和想法。最近发现很多技术博客博主会每年一个年度总结，可以看到他们的【一路走来】，可以看到他们的思考和成长，感觉这是一种很好的形式，不是为了给别人看，或许就是为了给自己看。

各种思绪、各种想法、做的好的，做的不好的。这些都值得被整理和记录。

我应该是不太会写这种东西，这种在我看来是务虚的文章。但是这个也不是考试，没啥会写不会写的，就想到哪写到哪，写就行了吧。

## 工作

工作上今年没有好事发生，一整年都没有调薪。

### 倦怠

工作上，今年并没有什么建树，或许不止今年，去年应该也没什么建树。

按部就班的写代码、按部就班的完成 OKR。并没有什么获得感、成就感。

完美主义的人就是这样，高标准、严要求，总觉得要轰轰烈烈才算结果。但是我只是一个大龄螺丝钉，我能做的或许就只有按部就班吧。

今年好像没有怎么大块的写代码，就是修修补补。手上的东西也在慢慢交给别人，除了 djob 还在深度的维护。

【你是来挣钱养家的，不是来拯救世界的】，当我感觉倦怠、感觉迷茫的时候，我总会想起来这句话。

有时候，我自信心爆棚，我觉得整栋楼就我技术最牛，我啥都会，啥都懂；跟哪个领域的同学都能唠几句。有时候，我又觉得，我好菜。一眼能看到未来几年的样子，就是一直是一颗写代码的螺丝钉，没法承接更大的 scope。

### 跳槽

有尝试过改变，有投过简历、面过试。我发现小厂面试没啥意思，面不到我的点上，给不了我酣畅淋漓的考试的感受。大厂面试还可以，面了一下大厂中间件，面试官还蛮专业，问的东西都专业，让我觉得我平时的积累没有浪费。

当然，面试最终没有通过，虽然是最后的 HR 面没过，但是应该也是之前的面试积分不高吧。

其实就是不会推销自己，面试的时候直来直去掏心掏肺的，想到哪说到哪。这个问题不光是面试的时候，其实在接触女生的时候也是。

找工作和找对象其实是差不多的事情，都是要包装自己、推销自己。

但是我不会这种，我太懒了。我总希望真实、自然的呈现。因为包装、掩饰这些对我来说都太费脑子了。我没有那个脑子，我总希望简单、真实、坦诚。

短期内应该不会再面试了，因为其实并没有想好，只是病急乱投医，觉得跳槽可以改变我的生活。但是我生活中的问题，只能是自己消化、解决。换个工作只是吃退烧药，但是病灶还在。

当然要是能有个涨幅 50% 的 offer，肯定也很好。思存说：树挪死人挪活，人有时候一直不顺利可能换个环境就好了。但是并没那么好找，我现在期待的薪资可能已经要面到 P8 才能给到，但是我的经历、背景是没法面到 P8 的，不是我技术上不行，我觉得技术上我肯定是够了，但是 P8 最看重的应该不是技术。必须要管理路线才能继续晋升也是让我很难受的一个点，我不会那块，当然我确实也没有花心思训练过那块。

### 滑铁卢

今年还遭遇了滑铁卢事件。

720 大故障最后的靴子落到了我身上，当然不光是我。

这个事情并不意外，当事情发生时我心理上就有准备的。当时想的是，塞翁失马焉知非福，要是扛雷拿个大礼包走，也不一定是坏事。但是最后是背 C 了，没被干掉，或许是因为这个事情涉及到的人，都算无辜吧，大家都是黑天鹅事件的受害者，包括公司。

720 事件给我的后劲很大很大。这个后劲不是因为 720 本身，而是这件事情让我感觉我的职业天花板到了，在这家公司干到头了。但是我太期待上升或者进步了，因为我对自己的价值判断就是看这些，看自己成家立业了没有、看涨薪了没有、看进步了没有。

当我感情不顺、生活不顺的时候，我就想失之东隅收之桑榆，期待在工作上有长进、有进步。

当工作不能承接我的期待的时候，那些已经被收拾好了的遗憾和恐惧，就排山倒海的冲向我，让我整个 Q3 都是消沉的。

很感谢 LD 和同事，看到我状态很差，给了我时间和空间。现在看，也是走出来了吧。尽管喝多了的时候还是会崩溃，但是日常上班的状态是回来了。

会有迷茫，迷茫的是找不到自己的价值，同时非常的没有安全感。

如果我一直是这样子写代码、只会写代码，那我的职业生命会很短，但是我除了写代码，别的都不会了。我也做不了技术管理。

我太理想主义了吧，我总感觉要代码写的很牛逼才能上升，太唯技术论了。看到别人博客里一段，非常启发我：

至于中间代码细节、方案细节，大家各展所长，各司其职就行了。没必要完美主义的、偏执的去指手画脚。

## 学习

因为失恋，所以一直在找事情分散注意力，所以今年花了很多时间学习。

### React

首先我掌握了 React，因为之前有用过 Vue，虽然就是会用，但是有这个基础，还是很快掌握了 React 的原理和基础开发。

React 是我唯一没有去系统的看书就学会的技能。因为按我的个性，掌握一个新课题一定是要去系统的啃几本书的，但是可能确实是比较简单吧 React，就看了看博客、文档就会了。

虽然不会写样式，但是用组件库写写后台管理页差不多够用了。

刚开始学这个的时候，总是会很伤感，突然伤感。尤其是见不得 Vue 字眼，因为之前加丰就是一天到晚学 Vue。

伤感的是：要是能问加丰就好了，加丰是搞前端的。但是又想到，如果我问他，我肯定也是逼逼赖赖，没有耐心，还会 cpu 他。

加丰肯定也不愿意搭理我。

前端也是蛮多细节的，加丰做的很好了已经，我之前也没有好好夸过他、肯定过他。他应该很期待我的肯定的，他太缺乏自信了，太努力的想被关注、关心了。

【我不是一个好哥哥】，近一年多来，我无数次的默念、咀嚼这句话。

都是遗憾，全是遗憾。

### K8S

我系统了刷了两本 K8S 的大部头，每本都是上千页。

K8S 是我一直想学但是没有系统去学的课题。虽然日常工作中没有直接的接触，但是我感觉这个东西很重要，是云原生时代的 Linux。

大概花了 2 个月的时间，看书、看博客、自己琢磨，建立了 K8S 的宏观认知和微观骨架。相比专业的云原生选手肯定是差一些，但是相对其他非专业选手，我现在的水平肯定是领先的。

现在让我去写 yaml 肯定是不熟练的，但是整体的架构、原理、技术路径我心里是有谱的。如果真的有需求，翻翻 API 文档肯定都能搞了。

### Linux 网络

我认真的看了几遍 Linux 网络原理，就一本很适合我的图文并茂的书，我认认真真的看了几遍，对网络这块的掌握更深刻了，不再是对着各种文字性的流程去脑补，而是有了代码层面自洽的认知。

甚至可以去看看 linux 协议栈代码片段，因为流于表面的、连环画式的讲解根本不能让我理解、自洽一些网络行为、原理。

一直想写一篇博客，用 Java 来基于协议栈讲述网络原理，但是这个课题太宏大了，牵涉的东西太多，所以一直没有动笔，26 年好好弄一下这个。

### Go

我认认真真的、耐心的学习了 Go。

尽管很多人跟我说，现在这个时候，去学一门新语言的 ROI 很低，因为有 AI 了。但是我还是觉得，AI 只能是助手，我自己得是专家，然后 AI 才能帮到我。

所以就像入行刚学 Java 那会一样，我耐心的开始看语法、看课程、刷书。每天晚上睡前复习自己白天的笔记，希望自己有扎实的 Go 基础。

为了学 Go 专门立了一个 OKR，12 月份一整个月都在带薪学 Go，然后开发了 djob-go-sdk。

到现在，应该是具备了初级的开发能力，因为写的少、看的多，所以写起来会觉得有些不顺手，这个后面再练练就好了。

总的来说已经过了新语言的阵痛期，慢慢的可以形成生产力了。

准备继续投入资源到 Go，一直到过完年，把基础搞扎实，后续多看看书、多看看开源代码。

我希望 Go 是我的第二母语，跟 Java 一样，可以如数家珍的那种。

### 博客

今年写了十几篇博客。

前几个月还把博客装修了下，现在的博客看起来更现代，之前的好丑。

写博客挺好的，找工作还真有用，面试官真的会看。

后面还是想继续写博客、做输出。但是不能太碎了，以后得开始写专栏。

再一个就是要转换自己的写作思路，不能老是面向 0 基础，以后写博客不能是为了教别人，而是为了让读者知道我懂、我会。

继续坚持吧。

后面如果生活稳定的话，一周出一篇博客吧。逐步的梳理、沉淀自己掌握的中间件、技术、架构。

## 生活

生活整体上还好吧，没有什么大的波澜。

### 妈妈脚骨折了

国庆后我妈脚踝骨折了，事情本来不大，因为 3 年前我也骨折过，差不多的位置，现在全好了，也没啥后遗症。

但是当时很紧张，害怕她那么大年纪有问题，也害怕问题很大，不是简单的骨折。尤其她怕我担心、分心，自以为没事还瞒了我两天我才无意中知道。

到现在已经可以正常走路了。

现在想想，其实都是小事情，只不过我就像是惊弓之鸟，杯弓蛇影的害怕家人出任何问题。但是我又只擅长呆在原地瞎想。

更多的麻烦其实是工伤那块，整个事情花不了多少钱，但是想给我妈讨个公道。中间差点去找律师起诉，但是最后还是嫌麻烦，或者是对我不信任我能找到的律师。总之最后是协商解决了。

我是没什么匪气的人，或者说，老实巴交的人。

遇到这些超出我认知的事情，比如找律师、跟我妈单位的人讨价还价，就会很焦虑，患得患失，不知道怎么处理。或许什么事情都有第一次，经历过了后面就懂了，有认知了。

但是，我的认知相对未知，永远都是更小的那部分，所以我常常会恐惧，恐惧变故、恐惧不确定性。

不管怎么样，尽管整个事情是不幸的意外，但是问题没有那么大，还是变成了生活中的一朵浪花。尽管在当时看来有点不知道怎么办，但是事情算是处理好了吧。

真的希望家人都好好的。

### 爸爸身体还好

我爸年纪大了，有些慢性病了开始。

但是有正儿八经的吃药控制，也有定期去医院复查。

希望我爸身体好好的，之前算命先生说，我爸只有 60 岁的寿命。之前都是听听笑笑，但是前段时间都很焦虑。

焦虑我都还没结婚、没生娃，还没有让他抱孙子。

现在身体也还好，那种焦虑慢慢的也平复了。

## 感情

做了一个很大的决定，失去了一个很依赖的人。

### 分手

情人节后几天分手的，我提的分手。

这对我来说不是一件容易的事情，但是现在再想想，真的也是缘分尽了。

没有谁不好、没有谁不对，就是缘分尽了，只能走到这里了，结束了两个人的痛苦。

会很舍不得，会很想念，分手之前各种吵架、争执，但是分手了能记起来的全是对方的好。但是啊，但是我太擅长压抑自己的感受了，因为也确实走不下去了。吵架、猜疑消耗了太多的感情和精力，猜疑的种子一旦种下，就只会长成参天大树，绝不会原地枯萎。

常常强迫性的回忆起她的温柔、她的体贴，她说过的话、做过的事、一起的经历。常常回想起自己哪里没有照顾好她、哪里欺负她了。一直到最近几天才好一些、平和一些，因为突然的意识到：是缘分尽了。

应该有个盖棺定论的，到现在快一年了。

她对我很好，我也很依恋她，让我狠下心分手的事情对于别人来说可能并不重要，或者对于以后的我来说可能也不重要。但是在当下，当那层窗户纸被我偏执的捅破后，事情就急转直下不能回头了。不用去纠结她好不好，也不用去纠结自己做的对不对，事情一路走到这里，就是缘分尽了。缘聚缘散，她是这样，加丰也是这样，所有的意难平、求不得都是这样。

大家都在往前走，她也在往前走，没有谁离不开谁的。她一定会很快的结婚、幸福的，跟我在一起只会让她难受、痛苦。她一定可以遇到比我更好、更成熟、更稳重的别人。她也一定有能耐把自己的生活经营的很好。

真诚的感谢她一年多的陪伴，她是个很称职的女朋友，给了我很多温暖、很多快乐，也支撑我度过低谷。

这段时间我想过很多，反思了很多，在一起的时候我把太多事情看成理所当然了，把对方的付出也看的太理所当然了。爱在当下，应该用心的、认真的用语言、行动表达自己的爱和关注，不用想那么多、想那么远，当下开心就尽情开心。

我想我以后会是一个更好的男朋友吧。

### 婚恋

今年也有陆陆续续的接触过几个女孩子，其实分手后的个把星期我就开始接触家里安排的女孩子，到现在见过面的女孩子有 4 个吧，都没有后续了。

刚分手的时候，我以为我可以什么都不想，什么都不管，随随便便就可以接受一个人。然后结婚，然后生娃。但是其实不行，跟对方点菜的时候、吃饭的时候、逛街的时候、各种时候，都会回想起前女友。

后来觉得不行，我还没有准备好接受下一段感情，就没有继续相亲了。

压力是有的，很大的压力。很长一段时间里都不敢早下班，或者早下班就去附近酒店过夜。因为爸妈还没休息，而我不敢回家面对父母。我妈总是忧心忡忡的看着我，欲言又止的问我要不要相亲。没有别的话题可以聊天了，几句话就会绕到相亲、结婚上。

后面也还有接触过几个女孩子，包括赵军看我状态那么差也是绞尽脑汁的帮我张罗，希望我赶紧好起来。

应该说我是挑剔的人，同时又是回避型人格，婚恋和亲密关系对我来说是很困难的一件事情。现在想想，上一段恋情之所以可以开始，应该全是因为她活蹦乱跳的、主动的、勇敢的闯入我的生活，不然我应该要母胎 solo 到今天。

依我的性格和目前的状态，我应该是很难进入婚姻的。我也不知道我是因为怎样的我不曾察觉的经历或阴影让我成了现在的样子。

我应该要尽快结婚的，尽快让家里热闹起来，这是我的责任。常常看着年迈的父母觉得不忍心，他们怎么这么不幸，摊上两个不争气的孩子。

这个问题，当下的我并不知道答案，但是我知道，这个答案绝对是等不来的。这件事是我目前最无力的一件事情。

## 身体

各种压力大，加上没有照顾好自己，身体一直亚健康。

### 膝盖不舒服

膝盖不舒服，从去年 Q3 就开始不舒服了左边膝盖。

应该是那会跑步机上爬坡跑步伤到膝盖了。

也有看过医生其实，在成都看的。医生说静养，慢慢会好的。

后来就在没有去看过医生了，有点讳疾忌医的意思。

我想肯定会好的吧，因为身体是有韧性的。我记得几年前我的腰就老是不舒服，我也是讳疾忌医的没管它，现在也好了。

这些都是小事情，我希望都是小事情。

### 高血脂/高胆固醇

五六年前胆固醇就高，近几年血脂也开始高了。

我一直不以为意，直到最近赵军说血脂高，我翻了翻体检报告，然后问了问大模型，发现这个事情不是小事情。

然后紧张到失眠，第二天一大早去看医生。

医生云淡风轻的，不像是什么大事情，给开了药。

印象中从我体检有脂肪肝开始这个指标就一直异常了，所以根因是内脏脂肪，所以应该好好运动、健身。
